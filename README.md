# ⚡ Zippity Studio — Content Creator Tool

A free, browser-based AI tool built for **Jerrica** and the **Zippity** brand.

Upload any product → get an AI-written Instagram caption, product description, and a downloadable animated promo video — all in seconds.

---

## ✨ Features

- 📸 Upload a product photo
- 🤖 AI generates product description + Instagram caption
- 🎬 Animated promo reel preview (Ken Burns effect, particles, glowing animations)
- ⬇️ Record & download the reel as a `.webm` video
- 🌙 Premium dark UI — no install, no account needed

## 🛠 How to Use

1. Open the live site
2. Upload your product photo
3. Enter the product name and a short description
4. Click **Generate content + promo video**
5. Copy your caption & description
6. Click **Record & download video** to save your promo reel

## 🔧 Tech Stack

- Pure HTML, CSS, JavaScript — no frameworks
- Anthropic Claude API (claude-sonnet) for AI content generation
- HTML5 Canvas for animated reel rendering
- MediaRecorder API for video download
